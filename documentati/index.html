Installation How to Install the script

Installation steps

Here's a brief information about how to install your scripts

  1. Download on your computer the ZIP file purchased from codecanyon.net.
  2. Extract its content in a local directory.
  3. Inside the folder extracted you will find three folders: documentation (contains the help file) and exmaple (contain demo)dima-switcher (contain the Dima - Skine Switcher itself) this is the folder you must upload to your server, and WP-dima-switcher for WordPress CMS.

    4. Ã— Note: Once you've downloaded and unzipped the Dima - Skine Switcher, open up the Dima - Skine Switcher folder. clicke and drag dima-switcher folder to your document root (i use : localhost/mywebsite/) .
  4. Include JS files into your pagejquery-1.10.2.min.js angular.min.js angular-cookies.js jquery.minicolors.min.js
  5. Include main JS files config.js ng-sw.js
  6. Include CSS files switcher-style-light.css for light version or switcher-style-dark.css for dark version and jquery.minicolors.css for Mini Colors plugin

WordPress Plugin Installation

  • With your FTP program, upload the Plugin folder "WP-dima-switcher" to the wp-content/plugins folder in your WordPress directory online.
  • Go to Plugins screen and find the newly uploaded Plugin in the list.
  • Click Activate Plugin to activate it.

Set Up quick start guide

Set up the script

1) Configuration
  1. Open config.js .
# Variables NAME DESCRPTION values
1 multipage show or hide multi page section true or false
2 onepage show or hide One page 'examples' section true or false
3 fullwidth show or hide Full width section true or false
4 boxedwidth show or hide boxed section true or false
5 skinStyle show or hide skin style section true or false
6 skinColor show or hide skin color section true or false
7 settings show or hide settings section true or false
8 showGrids show or hide Grids button true or false
9 box_image show or hide small images true or false
10 box_patterns show or hide small patterns true or false
11 sw_input show or hide input url true or false
12 floatBox where you need to show your switcher {left or right} left or right
13 timeout delete session cookies when time it out Integer
14 pluginsUrl Path to your plugin URL
2) How to config boxed and full-Width.
  1. Open config.js .
  2. Cahnge boxedClass = "boxed" to your css class ID that change your template from full-Width to boxed.
    Ã—Example: You have css class name ".makeThemeBoxed. the code will be like boxedClass = "makeThemeBoxed"
  3. Cahnge boxedInto = "body" to your your css class ID or html element where you need to apply boxed class.
3) How to config patterns and backgrounds images.
  1. Open config.js .
  2. Cahnge patternsInto = "body" to your css class ID or html element where you need to apply patterns.
    Ã—Example: You have css class name ".applPatterns. the code will be like patternsInto = ".applPatterns"
  3. Cahnge pgInto = "body" to your css class ID or html element where you need to apply patterns.
    Ã—Example: You have css class name ".applBgImage. the code will be like pgInto = ".applBgImage"
4) How to Add new patterns and background image.
  1. Upload patterns to dima-switcher\patterns\ and backgrounds images to dima-switcher\bg\
  2. In case you add backgrounds you need to create thumbnails images 40 X 40 px with name small-ChangeName.jpg
    Ã—Example: You have background name hello.jpeg,so you create thumbnail image and name it like this small-hello.jpeg
  3. Open config.js add pattern name into patt_images = [ .. ] and backgrounds images into bg_images = [ .. ] 
    4.    //List of patterns
    patt_images = [
        "New-Pattern.jpg",
    ],

    //List of images backgrounds
    bg_images = [
        "New-Background.jpg",
    ],
5) How to Add new css style file.
  1. Open config.js add style path into styles_css = [ .. ] 

    2.     //List of css styles
    styles_css = [
        ["Css Path ", "style name"],
        ["css/dark.css", "DARK"],
        ["css/light.css", "LIGHT"]
    ],
  2. Add data-them="" into your stylesheet link ,that you want to change after clicking dark or light version.
    Ã—Example: <link rel="stylesheet" data-them="" href="css/styles.css">
6) How to Add new new colors.
  1. Open config.js CSS colors (hex) into setColor = [ .. ]

  3. Listed all elements and css class ID you want to change background color properties and linking it with elementAndClassBg=" .. "
